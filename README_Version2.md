# NordMaterial - Website

Professional landing page for NordMaterial - electrical materials procurement service across EU.

## 🚀 Features

- **Modern Design**: Beautiful, responsive layout with dark theme
- **Contact Form**: Capture customer inquiries for price quotes
- **Mobile Responsive**: Works perfectly on all devices
- **Fast Performance**: Built with Next.js for optimal speed
- **SEO Friendly**: Optimized for search engines
- **Czech Language**: Full Czech localization

## 📋 Tech Stack

- **Next.js 14** - React framework
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Beautiful icons
- **Framer Motion** - Smooth animations (ready to use)
